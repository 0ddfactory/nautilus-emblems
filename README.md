# Nautilus Emblems

A file marking extension for Nautilus (GTK 4)

# INSTALL

In project directory:
```
chmod +x install.sh
sudo ./install.sh
nautilus -q
```

# UNINSTALL

In project directory:
```
chmod +x uninstall.sh
sudo ./uninstall.sh
nautilus -q
```

# EMBLEMS 

Nautilus Emblems will use any of these standard emblems:

 * `emblem-important`
 * `emblem-urgent`
 * `emblem-favorite`
 * `emblem-default`
 * `emblem-new`
